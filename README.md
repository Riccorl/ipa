## How to use

Install the library from [PyPI](https://pypi.org/project/nlp-toolkit):

```bash
pip install nlp-toolkit
```

or from [Conda](https://anaconda.org/Riccorl/nlp-toolkit):

```bash
conda install -c riccorl nlp-toolkit
```
